Operaciones de entrada/salida en JavaScript

Ejercicios

Ejercicio 1

Escriba el código JavaScript que sea necesario para:

  1. Leer un mensaje de texto desde un elemento de tipo input.
  2. Mostrar el mensaje de texto en la consola del navegador haciendo uso de la función console.log
